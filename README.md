Vocabulary Tracker:
A simple command-line tool to manage your personal vocabulary list. Add words, view them, search, and remove entries. Data is stored persistently in a JSON file.

Features:
Add new words with definitions
View all words (sorted alphabetically)
Search for exact word matches
Remove words
Persistent storage using JSON

Requirements:
Python 3.x

Installation:
Download or copy the script (vocabulary_tracker.py) to your computer.
Ensure Python is installed (run python --version to check).

Usage:
Run the script: python vocabulary_tracker.py
Follow the menu prompts:
1: Add a word
2: View all words
3: Search for a word
4: Remove a word
5: Exit
Data is saved to vocabulary.json in the same directory.

Notes:
Words are stored in lowercase for consistency.
The JSON file is created automatically on first use.
No external dependencies required.
